The core contracts depend on one utlity contract. Utility contracts only interact with core contracts.

| Utility Contract                                                                                                        | Description                                                                                                                                                                                                          |
|-------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [TokenRegistryOracle](https://github.com/EigenExplorer/liquid-avs-token/blob/dev/docs/utils/TokenRegistryOracle.sol.md) | Provides `LiquidToken` with latest prices of accepted assets during a user deposit. This helps the `LiquidToken` in determining the exact number of tokens that a user should receive based on the deposited assets. |
